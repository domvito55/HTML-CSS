Dell Inspirion 14" (5410)

Do you need a new computer for either accessing the Internet, write documents, or playing silly games? Consider to buy one of the many versions of the Dell Inpirion 14".

If you just need a personal computer (PC) for a day-to-day use, the version in conventional laptop format Intel® Core™ i3 should be enough for you.

If you are a student, I may consider to buy a 2-in-1 format (computer + tablet) so you can use it as a compuer and also take classnotes using a stylus. You can save some money buying the version with Intel® Core™ i5, 8 GB of RAM and 256 GB SSD. But if only $ 50.00, you can buy a much powerful machine withIntel® Core™ i7, 12 GB of RAM and 512 GB SSD.
